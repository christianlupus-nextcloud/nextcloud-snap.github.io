# Welcome

The official Nextcloud snap is designed to offer a fully production-ready instance of Nextcloud with minimal setup and maintenance (it updates automatically). If you aren't a web developer and don't need to tweak Nextcloud like one, the snap will probably work perfectly for you. The only thing we recommend you do up front is enable HTTPS. Run `nextcloud.enable-https -h` for more information.
